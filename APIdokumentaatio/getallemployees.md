# Näytä kaikki työntekijät

Näytä lista kaikista työntekijöistä. Kaikilla käyttäjäoikeuksilla saa etsittyä listauksen. 

**URL** : `/employees`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Data examples**

```json
    {
        "employeeid": 1,
        "firstname": "Liisa",
        "lastname": "Ihmemaa",
        "username": "user",
        "sales": []
    }
```
